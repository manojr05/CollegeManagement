# College Management System

This project is designed to store and manage data and records related of a college. It includes functionalities for managing students and teachers data.

## Features

- Three types of users: admin, student and teacher.
- Admin can view, add, delete, and modify student and teacher records.
- Teachers can track student marks and grades.
- Students can view their own details.
- Every user can login using their user-name and password.

## Getting Started

### Prerequisites

- Spring Tool Suite (STS) or Eclipse IDE
- Java JDK 8 or higher
- Maven
- MongoDB
- Postman

### Usage

- Access the Swagger UI documentation: [Swagger UI](http://localhost:8080/swagger-ui/index.html)
- Use Postman to test the APIs using the provided collection: [Postman Collection](./CollegeManagement.postman_collection.json)

### Database

- Create the database named "collegeManagement"
- Import the MongoDB collection: [Collection Export](./collegeManagement.user.json)
